 
# 🌌 Vocaglaxy

Vocaglaxy is an interactive vocabulary learning platform designed to help students improve English skills through quizzes, levels, animations, sounds, and educational challenges.

## 🚀 Features

- Vocabulary quizzes
- Multiple categories
- Real images
- Sound effects
- Leaderboards
- Levels and progress
- Firebase authentication
- Responsive design

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Firebase
- Netlify
